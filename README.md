# Freshly

This is a MEN Stack application for your fridge. No more spoiled food in your fridge!

## Technologies Used  
1. HTML, CSS, JavaScript
2. Node.js
3. Express
4. MongoDb, Mongoose
5. Axios and Edamam API
6. Bootstrap
7. ejs
8. Google OAuth and passport
9. Adobe Illustrator
